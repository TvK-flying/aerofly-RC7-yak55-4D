# aerofly-RC7-yak55-4D
a 4D yak55 for in aerofly RC7.

unzip the file and paste it in documents/aerofly RC 7/aircraft_mod.

now you open aerofly RC 7 and then choose the yak55 and select yak55 xxl 4D.

it still has a few problems: the prop does not brake, the prop animation when going backwards does not work and it has no engine sound.
